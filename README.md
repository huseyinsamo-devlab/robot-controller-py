# Robot Controller

This repository contains a Python-based robot controller for embedded systems. It includes motor control, sensor readings, and a simple web interface for remote operation.

## How to Use
1. Clone the repository.
2. Install dependencies from `requirements.txt`.
3. Run the main script using `python -m robot_controller.main --speed 50`.

## File Overview
- `src/`: Contains the main code.
- `tests/`: Contains test scripts.
